# 📚 Smart Scholar

Smart Scholar is a student financing application designed to help students manage their finances, gain insights through interactive dashboards, scan receipts, and explore job opportunities — all in one place.

---

## 🚀 Features

- 📊 Interactive financial dashboards
- 📧 Email insights and notifications
- 💼 Job opportunity explorer
- 🧾 Receipt scanning functionality
- 💡 Student-focused financial tracking and insights

---

## 🛠️ Tech Stack

### Frontend

- React
- Tailwind CSS

### Backend

- Node.js
- Express.js

### Email Service

- React Email
- Resend

---

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone <your-repo-url>
cd Alladin
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env` file in the root directory:

```env
RESEND_API_KEY="your_resend_api_key"

```

4. ### Start Email Service and Server

```bash
npm run email
```

```bash
node server.js
```

### 5. Start Frontend (on localhost)

```bash
npm run dev
```
